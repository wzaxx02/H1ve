# 题目：找到证据2

### 题目描述：在银行近的一次项目中，某个服务器的密钥被泄露，并被加密，密码是100-1000以内的质素总和。经过某种手段已经把嫌疑锁定，你能帮公司拿到关键性的证据吗

### 题目难度： 🌟🌟

### 维护：le31ei

### KEY: `flag{asdf%^&*ghjkl}`

### 配置信息： 
1. 开放端口： `8080`

### 解题过程：

1. 由题可知算出100-1000以内的质素总和为75067，它是1.zip的密码。
2. 将他放入winhex中查看，这里有一些字符，提取出来
3. 将这些字符转化为图片，发现是一个二维码
4. 扫描二维码出现答案
